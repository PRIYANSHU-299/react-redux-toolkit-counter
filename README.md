# ⚡ Redux Toolkit Dynamic Counter

A clean and interactive counter application that demonstrates modern global state management in React.



## 🛠️ Features
- **Increment/Decrement**: Basic ±1 operations.
- **Dynamic Input**: Set a specific number and update the state using `incrementByAmount`.
- **Global State**: State is managed via Redux Store (no prop drilling).
- **Responsive UI**: A simple, centered dashboard layout.

## 🚀 Tech Stack
- **Library**: React.js
- **State Management**: Redux Toolkit (Slices, Store, Hooks)
- **Hooks**: `useSelector`, `useDispatch`, `useState`
